# 🎮 Neon Rock Paper Scissors (Flask Game)

A visually stunning **Rock-Paper-Scissors web game** built using **Flask + HTML + CSS + JavaScript**, featuring a futuristic neon-themed UI and smooth interactions.

---

## 🌟 Features

* ⚡ Neon glowing UI (modern gaming look)
* 🤖 Bot vs Player gameplay
* 🎲 Random bot move using Python
* 📊 Live score tracking (Player vs Bot)
* 🎯 Instant result display
* 💻 Clean and responsive layout

---

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS, JavaScript
* **Library Used:** random (Python)

## 🎮 How to Play

1. Choose one option:

   * ✊ Rock
   * ✋ Paper
   * ✌️ Scissors

2. Bot will randomly select its move 🤖

3. Result will be displayed:

   * ✅ You Win
   * ❌ Bot Wins
   * 🤝 Draw

4. Score updates automatically

---

## 🧠 Game Logic

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
* Same choices → Draw
